# Simple Python Chatbot 🤖💬

This is a basic command-line chatbot built using Python and a pre-trained GPT-2 language model from Hugging Face's Transformers library. It runs locally, gives quick responses, and is perfect for fun experiments or learning how chat models work.

## 💡 Features
- Runs in your terminal
- Powered by GPT-2
- No server needed
- Easy to customize

## 🚀 How to Run

1. Make sure Python 3 is installed.
2. Install the required libraries:
   ```
   pip install transformers torch
   ```
3. Save the script as `chatbot.py` and run:
   ```
   python chatbot.py
   ```

Type your messages and see what the bot says. Type `quit` to end the chat.

## 📦 Dependencies
- transformers
- torch

## ⚙️ Customization Ideas
- Replace GPT-2 with a different Hugging Face model
- Save chat logs
- Add a simple GUI with Tkinter or Gradio
- Connect it to Telegram or Discord

## 🧠 Notes
GPT-2 is not perfect, and sometimes it gives weird or off-topic answers. But it’s still fun to play with! 😊
